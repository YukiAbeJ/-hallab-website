今日も「生きる」を支えていきます

予防・医療・介護のエキスパート集団による地域の健康づくり・予防・生活支援への新たな挑戦

Message

私たちは、
「生きる」を科学する
専門家集団です。

大学・研究機関、医療機関に所属し、臨床・教育・研究・社会実装の経験が豊富な研究者、臨床家らから構成される専門家集団です。

一次予防から医療、介護まで人々の生活を支援してきたリハビリテーションの専門家集団による、人々の「健康づくり、予防、生活支援」に係る新たな取組みです。

私たちは、予防・医療・介護のエキスパート集団であり、東京都立大学発のスタートアップとして、2025年4月に一般社団法人HAL Lab.を設立しました。
